# Weather Forecast Application

## Project Description
This is a Weather Forecast Application built using HTML, Tailwind CSS, and JavaScript.
The application fetches real-time weather data from an external API and displays
current weather conditions along with a 5-day forecast.

The UI is inspired by the reference image but designed differently to maintain originality.

## Features
- Search weather by city name
- Fetch weather using current location
- Five-column weather dashboard:
  - Temperature (with °C/°F toggle)
  - Feels like temperature
  - Humidity
  - Wind speed
  - Weather condition icon
- Extreme temperature alert for temperatures above 40°C
- Recent searched cities dropdown using localStorage
- 5-day extended forecast with weather details
- Dynamic background based on weather conditions
- Responsive design for desktop, tablet, and mobile
- Proper error handling without JavaScript alerts

## Technologies Used
- HTML5
- Tailwind CSS
- JavaScript (ES6)
- OpenWeatherMap API

## Setup Instructions
1. Download or clone the repository
2. Open the project in VS Code
3. Get an API key from OpenWeatherMap
4. Replace YOUR_API_KEY_HERE in script.js
5. Open index.html using Live Server
